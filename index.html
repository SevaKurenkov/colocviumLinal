01 · основа
Векторное пространство: определение и примеры

Понятное определение. Векторное пространство — это любая «математическая площадка» (множество объектов), на которой заданы два правила игры: как эти объекты складывать между собой и как их умножать на обычные числа. Чтобы эта площадка официально считалась векторным пространством, результаты этих действий не должны выходить за её пределы, а сами действия обязаны подчиняться 8 строгим законам (аксиомам).

Множество объектов обозначают буквой V. Сами объекты называют векторами (x, y, z). Числа, на которые умножают векторы, берутся из множества действительных чисел R и называются скалярами (α, β).

Правила операций.

  • Сложение. Берём два вектора x и y из V, складываем их и получаем новый вектор x + y, который тоже лежит в V.
  • Умножение на число. Берём вектор x из V и любое число λ из R. При умножении получается новый вектор λx, который остаётся внутри V.

8 аксиом.

  • Ассоциативность сложения: (x + y) + z = x + (y + z).
  • Нулевой вектор 0: 0 + x = x.
  • Противоположный вектор −x: x + (−x) = 0.
  • Коммутативность сложения: x + y = y + x.
  • Ассоциативность умножения на числа: (αβ)x = α(βx).
  • Дистрибутивность по числам: (α + β)x = αx + βx.
  • Дистрибутивность по векторам: α(x + y) = αx + αy.
  • Единица: 1·x = x.

Примеры. Mat(m×n) — множество всех прямоугольных матриц m×n. Rⁿ — обычное координатное пространство (строки или столбцы из n чисел). R[x] — множество всех многочленов (например, 5x² − 3x + 2), каждый многочлен — отдельный вектор.

02 · зависимость
Линейно зависимые и независимые системы

Понятное определение. Пусть есть набор векторов v₁, v₂, …, vₙ. Мы пытаемся собрать из них нулевой вектор 0, умножая каждый на число и складывая: α₁v₁ + α₂v₂ + … + αₙvₙ = 0. Это называется линейной комбинацией.

  • Линейно независимая система. Ноль можно получить только одним способом — если все коэффициенты равны нулю: α₁ = α₂ = … = αₙ = 0. Ни один вектор нельзя выразить через остальные.
  • Линейно зависимая система. Ноль можно собрать так, что хотя бы один коэффициент не равен нулю. Значит, есть «лишний» вектор, который выражается через остальные.

Связь «зависимость ↔ выражаемость».

В одну сторону. Если система зависима, существует ненулевая комбинация α₁v₁ + … + αₙvₙ = 0. Пусть α₁ ≠ 0. Тогда α₁v₁ = −α₂v₂ − … − αₙvₙ, и после деления на α₁ получаем явное выражение v₁ через остальные.

В другую сторону. Если, например, v₁ выражается через остальные: v₁ = c₂v₂ + … + cₙvₙ, перенесём всё в одну сторону: 1·v₁ − c₂v₂ − … − cₙvₙ = 0. Здесь коэффициент при v₁ равен 1, то есть не ноль. Значит, система линейно зависима.

Примеры.

  • ЛНЗ: единичные векторы (1, 0)ᵗ и (0, 1)ᵗ — направлены по разным осям, один через другой не получить.
  • ЛНЗ: 1, x, x² — из 1 и x нельзя сложить параболу x².
  • ЛЗ: многочлены x + 1, x − 1, 2, так как (x + 1) − (x − 1) = 2.
  • ЛЗ: два пропорциональных вектора v₁ и v₂, если v₁ = c·v₂ — один является растянутой копией другого.
03 · каркас
Базис векторного пространства

Понятное определение. Базис — это идеальный «каркас» пространства. Набор векторов e = {e₁, e₂, …, eₙ}, который:

  • не содержит лишних векторов (они линейно независимы);
  • достаточен для построения любого вектора пространства (любой вектор выражается через них единственным образом).

Количество векторов в базисе — это размерность пространства: dim V = n.

Теорема о разложении по базису. Набор векторов является базисом тогда и только тогда, когда любой вектор v можно единственным образом представить в виде v = x₁e₁ + x₂e₂ + … + xₙeₙ.

Доказательство (в одну сторону). Пусть {e₁, …, eₙ} — базис. Добавим к нему произвольный вектор v. Тогда система {e₁, …, eₙ, v} станет линейно зависимой. Значит, существует ненулевая комбинация α₁e₁ + … + αₙeₙ + κv = 0, где κ ≠ 0. Выражаем v через e₁, …, eₙ — разложение существует. Если бы было два разных разложения, вычитая их, мы получили бы ненулевую линейную комбинацию базисных векторов, равную нулю, что противоречит их независимости. Значит, разложение единственно.

Доказательство (в другую сторону). Если любой вектор раскладывается по системе e единственным образом, то:

  • ноль можно представить только как 0·e₁ + … + 0·eₙ, значит, других комбинаций, дающих 0, нет — система независима;
  • любой вектор выражается через e, значит, добавление любого нового вектора делает систему зависимой — e максимальна и является базисом.

Примеры базисов.

  • На плоскости V(E²) базис — (1, 0)ᵗ и (0, 1)ᵗ.
  • В Rⁿ стандартный базис — столбцы, где единица стоит по очереди на каждом месте, остальные нули.
  • В R[x] естественный базис — 1, x, x², …, xⁿ.
  • В Mat(m×n) базис — матричные единицы: матрицы с одной единицей и нулями в остальных местах.
04 · координаты
Координаты, матрица перехода, преобразование

Координаты вектора. Если v разложен по базису e = {e₁, …, eₙ} как v = x₁e₁ + … + xₙeₙ, то числа x₁, …, xₙ называются координатами вектора v в базисе e. Их удобно записывать столбцом x.

Матрица перехода. Пусть есть старый базис e и новый базис e'. Каждый вектор нового базиса eᵢ' можно разложить по старому базису e. Если координаты этих векторов записать столбцами и собрать в одну квадратную матрицу T, то получим матрицу перехода от e к e'. Кратко: e' = eT.

Преобразование координат. Сам вектор v не меняется, но его координаты зависят от базиса. В старом базисе: v = e x. В новом: v = e' x'. Приравниваем: e x = e' x' = e T x'. Отсюда x = T x' и x' = T⁻¹ x. То есть:

  • старые координаты через новые: x = T x';
  • новые координаты через старые: x' = T⁻¹ x.
05 · подпространство
Подпространство: определение и примеры

Понятное определение. Подпространство — это «пространство внутри пространства». Берём подмножество W из большого векторного пространства V. Оно будет подпространством, если:

  • сумма любых двух векторов из W снова лежит в W;
  • любой вектор из W, умноженный на любое число, остаётся в W.

Обязательно: в подпространстве должен лежать нулевой вектор 0. Если нуля нет, это не подпространство, потому что 0·x = 0 должен оставаться внутри множества.

Примеры.

  • W = {x ∈ Rⁿ | A x = 0} — множество всех решений однородной системы линейных уравнений.
  • W = R≤n[x] — многочлены степени не выше n. Сумма таких многочленов не выходит за пределы степени n.
  • W — множество всех верхнетреугольных матриц (aᵢⱼ = 0 при i > j). Сумма двух верхнетреугольных матриц снова верхнетреугольная.
  • Симметричные матрицы (Aᵀ = A). Сумма симметричных матриц тоже симметрична.
06 · оболочка
Линейная оболочка системы векторов

Понятное определение. Пусть есть набор векторов S. Мы берём все возможные линейные комбинации этих векторов: растягиваем, сжимаем (умножаем на числа), складываем во всех вариантах. Множество всех получившихся векторов называется линейной оболочкой системы S и обозначается ⟨S⟩.

Геометрически: линейная оболочка одного ненулевого вектора — прямая; двух непараллельных — плоскость. Линейная оболочка — наименьшее подпространство, которое содержит все векторы из S. Её можно также определить как пересечение всех подпространств, содержащих S.

Почему пересечение подпространств — подпространство. Пусть есть семейство подпространств Wᵢ и их пересечение W = ⋂ Wᵢ.

  • Если x и y лежат в W, то они лежат в каждом Wᵢ. Так как каждое Wᵢ — подпространство, x + y лежит в каждом Wᵢ, а значит, и в их пересечении W.
  • Если x ∈ W и λ — число, то x лежит в каждом Wᵢ, а значит, λx тоже лежит в каждом Wᵢ и, следовательно, в W.

Значит, пересечение подпространств — тоже подпространство.

07 · решения СЛАУ
Почему решения однородной системы — подпространство

Понятное объяснение. Пусть A — матрица размера m×n. Рассматриваем множество W всех векторов-столбцов x, для которых A x = 0. Это пространство решений однородной системы линейных уравнений. Докажем, что W — подпространство.

  • Нулевой вектор. A·0 = 0, значит, нулевой вектор — решение и лежит в W.
  • Сумма решений. Пусть A x = 0 и A y = 0. Тогда A(x + y) = A x + A y = 0 + 0 = 0. Значит, x + y — тоже решение и лежит в W.
  • Умножение на число. Пусть A x = 0 и λ — любое число. Тогда A(λx) = λ(Ax) = λ·0 = 0. Значит, λx — тоже решение и лежит в W.

Следовательно, множество решений однородной системы — подпространство.

08 · дополнение
Теорема о дополнении до базиса

Формулировка. Пусть V — конечномерное векторное пространство размерности n, и в нём есть линейно независимый набор векторов e₁, …, eₖ. Тогда k ≤ n. Если k < n, то к этим векторам всегда можно добавить новые eₖ₊₁, …, eₙ так, чтобы получился базис всего пространства V.

Понятное доказательство.

  • Если k = n, то по определению размерности набор уже является базисом, дополнять нечего.
  • Если k < n, рассмотрим линейную оболочку W = ⟨e₁, …, eₖ⟩. Векторы e₁, …, eₖ образуют базис в W, поэтому dim W = k. Так как k < n, W занимает только часть V. Значит, в V найдётся вектор eₖ₊₁, который не лежит в W.

Добавим eₖ₊₁ к системе. Получим {e₁, …, eₖ, eₖ₊₁}. Докажём, что она независима. Если бы она была зависимой, то eₖ₊₁ выражался бы через e₁, …, eₖ, то есть лежал бы в W, что противоречит выбору. Значит, система по-прежнему независима. Если векторов всё ещё меньше n, повторяем процесс, пока не получим n векторов. В итоге получаем базис.

Связь размерностей. Если W — подпространство в V, то dim W ≤ dim V. Причём dim W = dim V возможно только тогда, когда W совпадает со всем пространством V.

09 · сумма и пересечение
Сумма и пересечение подпространств

Понятное определение. Пусть в одном пространстве V живут два подпространства U и W.

  • Пересечение U ∩ W. Множество векторов, которые одновременно принадлежат и U, и W. Это всегда подпространство.
  • Сумма U + W. Множество всех векторов вида x + y, где x ∈ U, y ∈ W. То есть U + W = {x + y | x ∈ U, y ∈ W}.

Если U = ⟨S⟩ и W = ⟨T⟩, то U + W = ⟨S ∪ T⟩ — линейная оболочка объединения систем.

Формула Грассмана. Размерности связаны формулой:

dim(U + W) = dim U + dim W − dim(U ∩ W).

10 · отображение
Линейное отображение и его матрица

Понятное определение. Линейное отображение φ: V → W — это «трансформатор», который переводит векторы из V в W и уважает линейные операции:

  • φ(x + y) = φ(x) + φ(y);
  • φ(αx) = α φ(x).

Матрица отображения. Зафиксируем базис e = {e₁, …, eₙ} в V и базис f = {f₁, …, fₖ} в W. Для каждого eᵢ рассмотрим φ(eᵢ) и разложим его по базису f. Координаты φ(eᵢ) записываем столбцом. Склеив такие столбцы для всех i, получаем матрицу A размера k×n. Тогда действие φ на вектор v в координатах сводится к умножению матрицы на столбец: φ(v) соответствует A x.

11 · ядро и образ
Ядро и образ линейного отображения

Понятное определение. Для линейного отображения φ: V → W выделяют две ключевые зоны:

  • Ядро ker φ. Все векторы x ∈ V, которые переходят в ноль: φ(x) = 0. Это «черная дыра» отображения.
  • Образ Im φ. Все векторы w ∈ W, которые получаются как φ(v) для каких-то v ∈ V. Это «зона покрытия» в пространстве W.

Почему это подпространства.

  • Если x, y ∈ ker φ, то φ(x) = 0 и φ(y) = 0. Тогда φ(x + y) = φ(x) + φ(y) = 0, а φ(λx) = λφ(x) = 0. Значит, ker φ — подпространство в V.
  • Если w₁, w₂ ∈ Im φ, то w₁ = φ(v₁), w₂ = φ(v₂). Тогда w₁ + w₂ = φ(v₁) + φ(v₂) = φ(v₁ + v₂), а λw₁ = λφ(v₁) = φ(λv₁). Значит, Im φ — подпространство в W.

Теорема о ранге и дефекте. Размерность V раскладывается на размерность ядра и образа:

dim V = dim ker φ + dim Im φ.

12 · характеристический многочлен
Характеристический многочлен и собственные числа

Понятное определение. Линейный оператор — это линейное отображение из V в V. Его матрица A — квадратная. Характеристический многочлен χ_A(t) строится так: берём tE − A (E — единичная матрица) и считаем det(tE − A). Получаем многочлен степени n:

χ_A(t) = det(tE − A) = tⁿ − tr(A) tⁿ⁻¹ + … + (−1)ⁿ det A.

Здесь tr(A) — след матрицы (сумма диагональных элементов), а свободный член связан с det A.

Собственные числа. Собственное число λ — это число, для которого существует ненулевой вектор x, такой что A x = λ x. Вектор x называется собственным вектором.

Связь с характеристическим многочленом. Число λ является собственным числом оператора тогда и только тогда, когда оно является корнем характеристического многочлена: χ_A(λ) = 0.

13 · собственные векторы
Собственные векторы и метод их нахождения

Понятное определение. Собственный вектор — это вектор, который при действии оператора не меняет направления, а только растягивается или сжимается: A x = λ x, где λ — собственное число.

Метод нахождения.

  • Шаг 1. Находим собственные числа: составляем характеристический многочлен det(tE − A) и решаем уравнение det(tE − A) = 0. Его корни — λ₁, λ₂, …
  • Шаг 2. Для каждого λ решаем систему (A − λE)x = 0. Это однородная система линейных уравнений.
  • Шаг 3. Находим фундаментальную систему решений. Все ненулевые решения образуют собственное подпространство V_λ. Любой ненулевой вектор из V_λ — собственный вектор для числа λ.
14 · комплексные числа
Действия с комплексными числами, корни

Понятное определение. Чтобы извлекать корни из отрицательных чисел, вводят мнимую единицу i, для которой i² = −1. Комплексное число z = a + bi состоит из вещественной части a (Re z) и мнимой части b (Im z).

Основные действия.

  • Сложение и вычитание. (a + bi) ± (c + di) = (a ± c) + (b ± d)i.
  • Умножение. (a + bi)(c + di) = ac + adi + bci + bdi² = (ac − bd) + (ad + bc)i.
  • Деление. Чтобы разделить на a + bi, умножают числитель и знаменатель на сопряжённое a − bi. В знаменателе получается a² + b², а в числителе — новое комплексное число.

Извлечение квадратного корня. Ищем w = x + yi такое, что (x + yi)² = a + bi. Раскрываем квадрат: x² − y² + 2xy i = a + bi. Приравниваем вещественную и мнимую части:

x² − y² = a,  2xy = b.

Из второго уравнения y = b / (2x). Подставляем в первое, получаем уравнение относительно x, решаем его и затем находим y. В итоге получаем два комплексных корня.

15 · тригонометрическая форма
Тригонометрическая форма и формула Муавра

Понятное определение. Комплексное число z = a + bi можно представить как точку (a, b) на плоскости. Тогда удобно использовать полярные координаты:

  • модуль r = |z| — длина вектора от начала координат до точки;
  • аргумент φ — угол между этим вектором и положительным направлением оси x.

Из геометрии: a = r cos φ, b = r sin φ. Тогда z можно записать как:

z = r (cos φ + i sin φ).

Формула Муавра. Для возведения в степень n:

zⁿ = rⁿ (cos nφ + i sin nφ).

То есть модуль возводится в степень, а аргумент умножается на n.

16 · корни n-й степени
Корни n-й степени из комплексного числа

Понятное определение. Пусть дано комплексное число w = r (cos φ + i sin φ). Мы ищем все комплексные числа z, такие что zⁿ = w. В комплексной плоскости у корня n-й степени всегда ровно n различных значений.

Формула корней.

zₖ = √[n]{r} (cos((φ + 2πk)/n) + i sin((φ + 2πk)/n)),  k = 0, 1, …, n − 1.

Модуль берём с корнем степени n, а аргумент делим на n и добавляем 2πk/n. Геометрически корни лежат на окружности и образуют вершины правильного n-угольника.

Пример: z³ = −8.

  • Число −8 имеет модуль r = 8 и аргумент φ = π (стрелка смотрит влево). Значит, −8 = 8 (cos π + i sin π).
  • Модуль корней: √[3]{8} = 2. Аргументы: (π + 2πk)/3.
  • k = 0: угол π/3, z₀ = 2 (cos π/3 + i sin π/3) = 1 + √3 i.
  • k = 1: угол π, z₁ = 2 (cos π + i sin π) = −2.
  • k = 2: угол 5π/3, z₂ = 2 (cos 5π/3 + i sin 5π/3) = 1 − √3 i.
17 · формы
Билинейные и квадратичные формы

Понятное определение.

  • Билинейная форма β(x, y). Функция, которая берёт два вектора x и y и возвращает число. Линейна по каждому аргументу отдельно. Пример — скалярное произведение.
  • Квадратичная форма Q(x). Получается из билинейной формы, если подставить один и тот же вектор дважды: Q(x) = β(x, x). В координатах это выражение, где координаты вектора возводятся в квадрат и перемножаются (например, 3x₁² + 2x₁x₂ − 5x₂²).

Матрица формы. В выбранном базисе у векторов есть координаты x и y. Тогда билинейную форму можно записать как β(x, y) = xᵀ B y, где B — матрица формы. Для квадратичной формы Q(x) = xᵀ B x, при этом B обычно берут симметричной.

Смена базиса. Пусть e' = e T — переход к новому базису, а координаты меняются по x = T x', y = T y'. Тогда:

β(x, y) = xᵀ B y = (T x')ᵀ B (T y') = x'ᵀ (Tᵀ B T) y'.

Значит, новая матрица формы B' выражается через старую так: B' = Tᵀ B T.

18 · положительная определённость
Положительно определённые формы и критерий Сильвестра

Понятное определение. Квадратичная форма Q(x) называется положительно определённой, если для любого ненулевого вектора x выполняется Q(x) > 0, а Q(x) = 0 только для x = 0. Это означает, что форма «смотрит» строго вверх во всех направлениях, без нулевых и отрицательных направлений.

В каноническом виде такая форма имеет только плюсы перед квадратами координат.

Критерий Сильвестра. Пусть B — матрица квадратичной формы. Рассмотрим её ведущие главные миноры (угловые миноры):

  • Δ₁ — элемент на позиции (1, 1);
  • Δ₂ — определитель левого верхнего блока 2×2;
  • …;
  • Δₙ — det B.

Квадратичная форма является положительно определённой тогда и только тогда, когда все эти миноры положительны: Δ₁ > 0, Δ₂ > 0, …, Δₙ > 0.
